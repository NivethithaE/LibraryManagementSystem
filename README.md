# Library Management System

A console-based Library Management System built using Core Java.

## Features
- Add, search, issue, and return books
- Custom exception handling (BookNotFoundException, BookAlreadyIssuedException)
- Built using Collections (ArrayList)

## Concepts Used
- Object-Oriented Programming (Encapsulation, Classes & Objects)
- Java Collections Framework (ArrayList)
- Custom Checked Exceptions
- Exception Handling (try-catch, InputMismatchException)

## Tech Stack
- Core Java (no frameworks, no database, no GUI)

## How to Run
1. Clone the repository
2. Open in Eclipse (or any Java IDE)
3. Run `LibraryManagementSystem.java`

## Project Structure
```
src/com/library/
├── Book.java
├── Library.java
├── BookNotFoundException.java
├── BookAlreadyIssuedException.java
└── LibraryManagementSystem.java
```
